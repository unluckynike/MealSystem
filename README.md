# 网络点餐系统

## 项目信息

**系统环境**

- 开发环境：Windows
- 开发工具：Eclipse2019-9
- Java版本：JDK13
- 服务：Tomcat9.0
- 数据库：MySQL5.7
- 系统采用技术： JavaSE+Spring+SpringMVC+JSP+Servlet+Bootstrap
